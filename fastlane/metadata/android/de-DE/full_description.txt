Whisper ist ein Eingabemethodeneditor (IME) mit Spracherkennung basierend auf Whisper

Es kann als eigenständige App verwendet werden und bietet auch einen IME, der z. B. über die Mikrofontaste in HeliBoard aktiviert werden kann.

Beim ersten Start müssen die Whisper-Modelle von Hugging Face heruntergeladen werden.

Dies ist die einzige Verwendung der Internetberechtigung.

Die Spracherkennung erfolgt vollständig offline.

So holen Sie das Beste aus Whisper heraus:
- Halten Sie die Taste gedrückt, während Sie sprechen.
- Machen Sie eine kurze Pause, bevor Sie mit dem Sprechen beginnen.
- Sprechen Sie deutlich, laut und in moderatem Tempo.
